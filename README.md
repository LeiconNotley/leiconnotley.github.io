# Leicon Notley Website

Official website for Leicon Notley.

## Structure

* `index.html` – main page
* `images/clients` – external logos
* `images/favicon` – favicon
* `images/hero` – section hero images
* `images/iso` – HSEQ assets
* `images/logo` – company logos
* `images/pjojects` – project images
* `css/ – style sheet
* `js/ – javascript

* `robots.txt`, `sitemap.xml` – basic SEO files

## Deployment

Upload the entire folder contents to your web root.

## Notes

* The page references Google Fonts and Font Awesome via CDN (no build step required).
* iPhone Safari viewport fixes are included (uses `100svh` fallback from `100vh`).

## Features

- Responsive design
- Mobile optimized
- Lazy-loaded project galleries
- Client carousel
- Project filtering
- Contact floating action button
- SEO optimized

## Performance Optimizations

- Lazy loaded project images
